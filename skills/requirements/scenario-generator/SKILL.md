# Scenario Generator

## Purpose

Generate use case scenarios and edge cases from requirements.

## Workflow

1. **Input**: Requirements.
2. **Extract**: Identify actors, flows, conditions.
3. **Generate**: Produce scenarios (happy path, alternatives, edge cases).
4. **Output**: Structured scenarios (e.g., use case format).

## Output

- Use case scenarios
- Edge cases and alternatives
- Format suitable for test design
